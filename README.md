## Installation

### Clone repo
``` bash
# clone the repo
$ git clone https://github.com/mecdcme/istatboard.git istatboard

# go into app's directory
$ cd istatboard

# install app's dependencies
$ npm install
```

## Usage

``` bash
# serve with hot reload at localhost:4200.
$ ng serve

# build for production with minification
$ ng build
```
